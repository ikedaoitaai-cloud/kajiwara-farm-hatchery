# 孵化率予測システム - プロジェクト概要

## 📋 プロジェクト情報

**クライアント**: カジワラファーム
**目的**: 孵化率予測の精度向上とシステム化
**現状**: お母様の手計算（精度70-75%）  
**目標**: システム化により精度85-90%へ改善

## 🧮 予測アルゴリズム詳細

### 区分線形モデル（推奨）
お母様の経験値ベース、理解しやすく調整が容易
```
f(age) = {
  20                                          (age < 165)
  20 + (40-20) × (age-165)/(230-165)         (165 ≤ age ≤ 230)
  40                                          (230 < age ≤ 250)
  40 - (40-35) × (age-250)/(530-250)         (250 < age ≤ 530)
  35 - (35-32) × min((age-530)/200, 1)       (age > 530)
}
```

**特徴:**
- 若すぎる親鳥（165日以下）: 孵化率は20%程度
- 成長期（165-230日）: 孵化率は線形に上昇
- ピーク期（230-250日）: 孵化率は40%で、これがピーク
- 成熟期（250-530日）: 孵化率は緩やかに下降
- 高齢期（530日以上）: 孵化率はさらに下降

### 非対称正規分布モデル
生物学的モデル、滑らかな変化
```
f(age) = 24.5 + 15.5 × exp(-0.5 × ((age-240)/σ)²)

σ = { 32  (age ≤ 240)  ← 左側急峻
      300 (age > 240)  ← 右側緩やか
```

**特徴:**
- 孵化率ピーク位置: 240日齢前後
- 孵化率ピーク値: 40%
- 左右非対称（若い方が急激に変化）

### 補正係数

**貯卵日数補正:**
- 7日以内: 1.0（補正なし）
- 8-10日: 0.95（5%低下）
- 11-16日: 0.85（15%低下）
- 17日以上: 0.60（40%低下）

**季節補正:**
- 春・秋（3-5月、10-12月）: 1.0
- 夏期（6-9月）: 0.92（暑さによる卵劣化）
- 正月明け（1月）: 0.95（孵卵器温度立ち上がり時間）
- 冬期（2月）: 1.0

**メス鶏率:**
- 固定値: 61%（オス39%）

**最終計算式:**
```
孵化率 = 基礎孵化率(親鳥日齢) × 貯卵日数補正 × 季節補正
メス鶏数 = 総孵化数 × 0.61
```

## 🐛 既知の制限事項

### Phase 1（現在）
1. **LocalStorage依存**
   - ブラウザキャッシュクリアでデータ消失リスク
   - ブラウザ・デバイス間でデータ共有不可
   - → Phase 2でクラウドDB対応予定

2. **認証なし**
   - 誰でもアクセス可能（URLを知っていれば）
   - → Phase 2で認証機能追加予定

3. **バックアップなし**
   - 自動バックアップ機能なし
   - → 手動エクスポート機能を追加予定

4. **オフライン非対応**
   - インターネット環境必須（CDN依存）
   - → PWA化で改善検討

## 🤖 ML モデル開発計画（Phase 4）

### 6ヶ月時点での中間評価項目
1. **予測精度の現状把握**
   - MAPE（平均絶対誤差率）の計算
   - 区分線形 vs 非対称正規分布の精度比較
   
2. **特徴量の重要度分析**
   - 親鳥日齢の影響度
   - 貯卵日数の影響度
   - 季節要因の影響度
   - 農場間差異の有無
   
3. **ML導入の費用対効果試算**
   - 現状予測式との精度差
   - 改善見込み vs 開発コスト
   - ROI計算

### ML モデル候補

**レベル1（簡易・解釈性重視）:**
- 線形回帰 + 多項式特徴量
- 実装容易、計算高速
- 解釈性が高い

**レベル2（推奨・バランス型）:**
- XGBoost / LightGBM
- 高精度、過学習制御可能
- 特徴量重要度が取得可能

**レベル3（高度・精度重視）:**
- ニューラルネットワーク
- 最高精度の可能性
- データ量200+サンプル必要

### 特徴量エンジニアリング案
```python
# 基本特徴量
- parent_age          # 親鳥日齢
- egg_count           # 入卵数
- storage_days        # 貯卵日数
- month               # 月
- farm_name           # 農場名

# 派生特徴量
- age_squared         # 日齢の2乗
- storage_log         # log(貯卵日数+1)
- age_storage_interaction  # 日齢×貯卵日数
- season_category     # 季節カテゴリ
- is_summer           # 夏フラグ
- is_january          # 1月フラグ
- days_from_peak      # ピーク日齢(240)からの距離

# ラグ特徴量（時系列）
- prev_hatch_rate     # 前回の孵化率
- avg_hatch_rate_last_3  # 直近3回の平均
```
